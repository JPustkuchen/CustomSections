######################### Custom Section type folders ##############################

Naming folders:

Since folder names here will automatically be used as section keys,
avoid .(dot), #(hash), brackets, parentheses and any sort of 
special characters like umlauts, accents, etc. These will cause errors!

Only use ASCII letters A-Z, a-z, _(underscore), -(minus sign/dash) and 0-9(numbers)

To disable a section type, prefix it's folder name 
with a !(exclamation mark) like !my_section
